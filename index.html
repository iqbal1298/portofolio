E Rekomtek BBWS Pena

Saya membuat dan mengembangkan E Rekomtek BBWS Pemali Juana pada saat Magang. Website tersebut dikembangkan dengan menambahkan fitur Tracking untuk melacak progres permohonan rekomtek. Saya menggunakan Framework PHP, CSS, dan Javascript dengan menerapkan algoritma Sequential Search.

Sistem Informasi Geografis TPI Pati adalah proyek yang saya kerjakan untuk client, yaitu kawan saya sendiri. Sistem ini berisi informasi letak titik TPI yang ada di Kabupaten Pati. Sistem ini dibuat dengan framework CI, Javascript. Dengan menerapkan Algoritma Haversine untuk menghitung jarak lintang antar titik.

Framework CodeIgniter, Javascript, CSS, bootstrap, OSM
